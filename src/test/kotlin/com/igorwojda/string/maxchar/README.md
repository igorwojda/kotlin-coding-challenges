# Max occurrent char

## Instructions

Given a string implement a function which returns the character that is most commonly used in that string.

[Challenge](Challenge.kt) | [Solution](Solution.kt)

## Examples

```kotlin
maxChar("abcccccccd") // "c"

maxChar("apple 1231111") // "1"
```
