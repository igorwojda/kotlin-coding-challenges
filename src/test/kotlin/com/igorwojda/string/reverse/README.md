# Reverse string

## Instructions

Given a string implement a function which returns a reversed copy of that string.

[Challenge](Challenge.kt) | [Solution](Solution.kt)

## Limitations:

Don't use Kotlin `String.reverse` function

## Examples

```kotlin
reverse('apple') == 'leppa'

reverse('hello') == 'olleh'

reverse('Greetings!') == '!sgniteerG'
```
