# Find the vowels

## Instructions

Given string implement a function, which returns the number of vowels used in a string. Vowels are the characters `a`
, `e` `i`, `o`, `u` and
`y`.

[Challenge](Challenge.kt) | [Solution](Solution.kt)

## Examples

```kotlin
vowels('Hi There!') // 3

vowels('Why do you ask?') // 6

vowels('Why?') // 1
```
