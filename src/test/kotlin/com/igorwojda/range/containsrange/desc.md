# Contains range?

## Instructions

Given two ranges implement a function which checks if range1 contains range2.

[challange](challange.kt) | [solution](solution.kt)

## Examples

```kotlin
containsRange(5..7, 5..7) // true

containsRange(1..12, 5..7) // true

containsRange(5..8, 5..9) // false
```
