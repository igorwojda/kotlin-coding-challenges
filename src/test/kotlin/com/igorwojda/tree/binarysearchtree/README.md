# Binary search tree

## Instructions

Implement [binary search tree (BST)](https://en.wikipedia.org/wiki/Binary_search_tree).

[Challenge](Challenge.kt) | [Solution](Solution.kt)

## Examples

```kotlin
//----------Tree------------
//
//           6
//         /   \
//        3     8
//       /
//      1   
//
//--------------------------

val bst = BinarySearchTree<Int>()
bst.add(6)
bst.add(3)
bst.add(1)
bst.add(8)
```
